====================================================
  PICK STAFFING EVALUATOR
  Live Dashboard for Pick Operations
====================================================

HOW TO START:
  1. Make sure Firefox is open and logged into FCLM
     (visit https://fclm-portal.amazon.com once)
  2. Double-click "Start Dashboard.bat"
  3. Dashboard opens automatically in your browser

That's it.

====================================================
OPTIONAL (for real-time active picker tracking):
  - Have Rodeo open with OB Pick Center v3.5+
  - This shows who is CURRENTLY PICKING vs inactive

====================================================
REQUIREMENTS:
  - Python 3.10+ (https://python.org/downloads)
  - Firefox browser (for FCLM authentication)
  - Dependencies auto-install on first run

====================================================
FEATURES:
  - Live UPH rates per process path (auto-refreshes)
  - Individual associate rates and performance
  - Cross-training permissions (verified from FCLM)
  - Staffing move recommendations (14-day history)
  - Active picker tracking (via Picking Console)
  - Editable goals per path
  - Night/Day shift selector

====================================================
TROUBLESHOOTING:
  
  Dashboard won't load:
    -> Make sure Firefox is open + logged into FCLM
    -> Refresh your FCLM session if expired

  X-Train not showing:
    -> Wait ~90 seconds after startup (loading in background)

  Shows stale data:
    -> Click "Refresh Now" or Ctrl+Shift+R

  "No Data" showing:
    -> Check that the shift time range is correct
    -> Click Reset to snap to current shift window
